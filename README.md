# 🏥 YakSok (약속)
> **AI 기반의 지속가능한 의료 물류 최적화 및 ESG 리포팅 솔루션**
> 
> *SMWU AI Hackathon 2025*

## 🌟 프로젝트 개요
**YakSok**은 중소 병원의 반복적인 재고 관리 업무를 자동화하고, AI 수요 예측을 통해 의료 폐기물을 최소화하여 경제적 효율성과 환경적 가치를 동시에 실현하는 경량화 SaaS 솔루션입니다.

## ✨ 핵심 차별화 포인트
1. **ESG 경영 지표 통합**: 단순 재고 관리를 넘어, 과잉 발주로 인한 폐기물 발생량을 CO₂ 배출량으로 환산하여 정량적 ESG 리포트를 제공합니다.
2. **중소병원 맞춤형 경량화**: 복잡한 ERP 시스템 없이 CSV 업로드만으로 즉시 AI 분석 및 예측 결과를 확인할 수 있습니다.
3. **지속가능한 생태계 구축**: 외부 보건 이슈(WHO, 보건복지부 등)와 연동하여 발주 권장안을 제시하며, 인근 병원 간 유효기간 임박 의약품 교환/나눔 인사이트를 제공합니다.
4. **생성형 AI 리포트**: 머신러닝의 정량적 데이터를 LLM이 스토리텔링 형식으로 요약하여 의사결정권자에게 최적의 대책을 제안합니다.

## 🛠 주요 기능
- **📊 데이터 시각화**: 재고 소진 추이 및 진료과별 사용 비중 대시보드.
- **📈 AI 수요 예측**: 과거 사용량과 수술/환자 추이 데이터를 결합한 품목별 미래 사용량 예측.
- **🚨 긴급 발주 알림**: 재고 부족 위험 품목 자동 탐지 및 최적 발주 수량 제안.
- **🌱 ESG 리포팅**: 환경적 영향(탄소 저감 등) 분석 및 지역사회 상생 나눔 제안.

## 📂 기술 스택
- **Backend**: Python, Flask, Pandas, Scikit-learn
- **AI/LLM**: Google Gemini API (or OpenAI GPT-4o)
- **Frontend**: React (or Streamlit)
- **Data**: CSV-based lightweight parsing

## 🚀 시작하기
```bash
# 가상환경 설정
python -m venv venv
source venv/bin/activate

# 필수 패키지 설치
pip install -r requirements.txt

# 서버 실행
python app.py
```
